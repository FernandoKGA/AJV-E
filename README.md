# AJV-E
Another JSON Validator - Examples

In this repository are some examples of validations with ajv and ajv-errors custom error messages.
They're using the `json-schema-draft-06`. Check the [JSON Schema Specification](http://json-schema.org/specification.html) and the packages repositories for more information.

Needed packages:
* [ajv](https://github.com/epoberezkin/ajv)
* [ajv-errors](https://github.com/epoberezkin/ajv-errors)
